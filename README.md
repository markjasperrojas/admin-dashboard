# admin-dashboard

This website is like a mini version of my portfolio. It was built from scratch and is based on a project from [The Odin Project](https://www.theodinproject.com/). This is the most challenging design I have built so far, but through this project I was able to improve my skills and gain more experience with CSS Grid and Flexbox layouts. The website is not yet responsive.

All of the icons and more can be downloaded as SVGs from [Material Design Icons](https://pictogrammers.com/library/mdi/).
